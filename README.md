## Description
This repository contains the source code for F-XGBoost, an XGBoost variant engineered with a focal loss objective function to address class imbalance in network intrusion detection. Evaluated on the CICIDS2017 dataset with SHAP explainability.

## Model Engineering
- Operation: REPLACE (log-loss → focal loss)
- Engineered model: F-XGBoost (Focal-XGBoost)
- Baselines: Standard XGBoost, Random Forest, LightGBM
